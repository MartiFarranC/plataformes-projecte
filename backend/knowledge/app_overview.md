# PostureGuard Chat Knowledge

This document provides background information for the posture assistant and helps the RAG system answer posture-related questions.

## App purpose

PostureGuard monitors ergonomic posture using camera and sensor data. The chat assistant can explain posture advice, correct common mistakes, and provide general guidance.

## Supported topics

- Ergonomic posture while using mobile devices
- Sensor-based posture detection concepts
- Common posture mistakes and how to fix them
- How to maintain healthy posture during work or study sessions

## Using the chat assistant

- Ask about posture correction tips.
- Ask how to recognize bad posture.
- Ask what to do if your neck or shoulders feel tense.

## Answer style

- Keep responses short and practical.
- Use friendly, direct advice.
- Reference the posture principles in the knowledge content.
